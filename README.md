# OpenClaw Skills & Projects

This workspace contains OpenClaw agent skills and demo projects.

## Skills

### UI/UX Pro Max
Professional UI/UX design intelligence with 67 styles, 96 palettes, and comprehensive design guidelines.
- **Location**: `skills/ui-ux-pro-max/`
- **Features**: Design systems, color palettes, typography, charts, accessibility guidelines
- **Usage**: Run `python3 skills/ui-ux-pro-max/scripts/search.py "<query>" --design-system` to get recommendations

## Projects

### Crypto Exchange Demo
A professional cryptocurrency trading platform with dark theme.
- **Location**: `crypto-exchange/`
- **Features**: Real-time charts, trading interface, order book, market overview

## Getting Started

For UI/UX design recommendations:
```bash
cd skills/ui-ux-pro-max
python3 scripts/search.py "your project type" --design-system
```
