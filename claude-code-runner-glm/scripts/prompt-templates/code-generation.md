# 代码生成提示模板

## 基础模板

```
你是专业的[技术栈]开发工程师。

任务：[具体任务描述]

技术要求：
1. 使用[具体技术/版本]
2. [其他技术要求]

代码规范：
- 函数不超过50行
- 变量命名清晰易懂
- 添加中文注释
- 完善的错误处理

文件保存到：[路径]

直接输出代码，不要markdown格式，不要代码块标记。
```

## 示例：生成3D游戏

```
你是专业的Three.js游戏开发工程师。

任务：创建一个3D网页游戏 - 可爱小孩吃蛋糕

技术要求：
1. 使用 Three.js r128 (从CDN加载)
2. 经典JavaScript (非ES模块)
3. 单HTML文件，直接双击打开

游戏功能：
1. 可爱的3D小孩角色（粉色圆球身体、大眼睛、微笑）
2. 点击地面移动小孩
3. 蛋糕随机出现（粉色蛋糕造型，有蜡烛）
4. 小孩吃到蛋糕时得分
5. 得分显示在屏幕上方

代码规范：
- 函数不超过50行
- 变量命名清晰易懂
- 中文注释
- 完善的错误处理

文件保存到：~/Desktop/my_game.html

直接输出HTML代码，不要markdown格式，不要代码块标记。
```

## 示例：生成Python脚本

```
你是专业的Python开发工程师。

任务：创建一个Python脚本来[任务描述]

要求：
1. 使用Python 3
2. 添加类型注解
3. 函数不超过50行
4. 中文注释
5. 完善的错误处理

文件保存到：~/Desktop/my_script.py

直接输出Python代码，不要markdown格式。
```
